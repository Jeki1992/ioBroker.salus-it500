# ioBroker.salus-it500

## Адаптер Salus IT500 для ioBroker

Цей адаптер дозволяє керувати та моніторити ваш термостат Salus IT500 через ioBroker.

## Можливості

- Читання поточної температури в приміщенні
- Читання та встановлення цільової температури
- Відстеження статусу опалення (увімкнено/вимкнено)
- Перемикання між режимами AUTO та OFF
- Автоматичне опитування статусу пристрою
- Управління сесією з автоматичним повторним входом

## Конфігурація

1. Встановіть адаптер з адміністративного інтерфейсу ioBroker
2. Налаштуйте адаптер, використовуючи облікові дані Salus IT500:
   - **Ім'я користувача/Email**: Email вашого облікового запису Salus IT500
   - **Пароль**: Пароль вашого облікового запису Salus IT500
   - **Інтервал опитування**: Як часто оновлювати дані пристрою (мінімум 30 секунд, за замовчуванням 60 секунд)

## Стани

Адаптер створює наступні стани:

### Стани керування
- `salus-it500.0.setTemp` - Цільова температура (читання/запис, °C)
- `salus-it500.0.autoMode` - Перемикач режиму AUTO (читання/запис, true=AUTO, false=OFF)

### Стани статусу
- `salus-it500.0.currentTemp` - Поточна температура в приміщенні (тільки читання, °C)
- `salus-it500.0.heatStatus` - Статус опалення (тільки читання, true=опалення працює, false=вимкнено)

### Інформаційні стани
- `salus-it500.0.info.connection` - Статус підключення (тільки читання)
- `salus-it500.0.info.deviceId` - ID пристрою (тільки читання)

## Приклади використання

### Встановлення цільової температури
```javascript
setState('salus-it500.0.setTemp', 22);
```

### Увімкнення режиму AUTO
```javascript
setState('salus-it500.0.autoMode', true);
```

### Вимкнення режиму AUTO (ручне керування)
```javascript
setState('salus-it500.0.autoMode', false);
```

### Читання поточної температури
```javascript
const currentTemp = getState('salus-it500.0.currentTemp').val;
console.log(`Поточна температура: ${currentTemp}°C`);
```
